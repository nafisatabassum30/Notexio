Notexio — a clean, modern text editor built in Python.

A lightweight, customizable editor with file safety features, formatting controls, themes, and productivity tools—implemented with Tkinter.

Tech: Python + Tkinter
Focus: UX + OS I/O
Highlights: autosave/recovery, theming
Watch demo (2–5 min) Read STAR report

What I’m showcasing

  • System design: feature modules instead of a monolith
  • UX details: shortcuts, status bar, line numbers, zoom
  • Reliability: unsaved-change protection + recovery workflow

Repo entry point

python main.py

Features

File operations

New/Open/Save/Save As + recent files + unsaved detection.

Edit tools

Undo/redo, find/replace, go-to-line, clipboard actions.

Formatting & themes

Font controls, bold/italic/underline, light/dark mode.

View controls

Zoom in/out/reset, fullscreen toggle, word wrap, line numbers.

Productivity tools

Word/line/character stats, reading time, duplicate highlighting.

Safety features

Recovery file creation + restore prompt on startup.
