## What Notexio Demonstrates (OS-Relevant)

- **File I/O + encoding**: open/read/write files with UTF-8.
- **State consistency**: unsaved-change detection and prompts.
- **Crash recovery**: recovery snapshots and restore-on-start.
- **Concurrency safety**: auto-save loop in a background thread (UI stays responsive).
- **Cross-platform behavior**: OS-specific scrolling + printing pipeline.

## Features (Short)

- New/Open/Save/Save As + Recent files
- Undo/Redo, Find/Replace, Go-to-line
- Zoom, Fullscreen, optional Line numbers
- Light/Dark/Custom themes
- Tools (counts, reading time, cleanup)
- Print + Export as PDF

## How to Run

```bash
pip install -r requirements.txt
```
